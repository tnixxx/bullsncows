# Игра «Быки и коровы»

[English version](README.en.md)

Классическая логическая игра «Быки и коровы» (вариант игры Mastermind) с графическим интерфейсом на Python и Tkinter.  
Цель игры — отгадать загаданное компьютером число за минимальное количество ходов.

![Скриншот игры](screenshot.png)

## Возможности

- Выбор длины загадываемого числа (от 3 до 6 цифр)
- Возможность разрешить или запретить повтор цифр
- Современный интерфейс с поддержкой светлой и тёмной темы (библиотека `sv-ttk`)
- Таблица истории ходов с чередованием цвета строк
- Прогресс-бар, показывающий текущий прогресс угадывания
- Подсказки (открывается одна случайная позиция)
- Авторешение (компьютер сам показывает загаданное число)
- Статистика игр (количество игр, побед, среднее число ходов, лучший результат, использованные подсказки и авторешения)
- Сохранение настроек и статистики между запусками (JSON-файлы)
- Поддержка двух языков: русский и английский (переключение в настройках)

## Технологии

- Python 3.14 (или 3.10+)
- Tkinter (входит в стандартную поставку Python)
- `sv-ttk` — современная тема для ttk
- PyInstaller (для сборки в .exe)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/tnixxx/bullsncows.git
cd bullsncows
```

---

### 2. Установка зависимостей
Убедитесь, что у вас установлен Python (версия 3.10 или выше). Затем установите библиотеку `sv-ttk`:

bash
pip install sv-ttk
Для сборки исполняемого файла также потребуется PyInstaller:

```bash
pip install pyinstaller
```
### 3. Запуск игры
```bash
python main.py
```
### 4. Сборка в .exe (для Windows)
```bash
pyinstaller --onefile --windowed --name "BullsAndCows" main.py
```
Готовый файл `BullsAndCows.exe` появится в папке `dist`.

### Использование
1. При запуске генерируется новое число.
2. Введите своё предположение в поле ввода и нажмите «Проверить» или клавишу Enter.
3. В таблице появится запись с количеством «быков» (цифры на своих местах) и «коров» (цифры, присутствующие в числе, но не на своих местах).
4. Продолжайте, пока не угадаете все цифры.
5. При победе появится поздравление и статистика обновится.

### Настройки
В меню «Игра» → «Настройки» можно изменить:

- Длину числа (3–6)

- Разрешить/запретить повтор цифр

- Тему оформления (светлая / тёмная)

- Язык интерфейса (русский / английский)

### Структура проекта
`main.py` — точка входа

`gui.py` — графический интерфейс (главное окно, диалог настроек)

`game_logic.py` — логика игры (генерация числа, проверка ходов, подсказки)

`settings.py` — управление настройками (сохранение/загрузка)

`stats.py` — управление статистикой

`i18n.py` — модуль локализации (поддержка русского и английского языков)

`settings.json` — файл с настройками (создаётся автоматически)

`stats.json` — файл со статистикой (создаётся автоматически)
